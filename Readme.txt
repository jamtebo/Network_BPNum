全连接神经网络：输入图片大小：28 X 28 识别图片中的数字
只支持单个数字

***********************文件功能**************************

image文件夹：把自己画的图片放里（未转成28X28大小）。
mnidata文件夹：MNIST数据集。
Weight文件夹：保存权重（训练时间太长了）。
ImageOperate.py：图片操作类，将图片转成28X28。
MNIST_TRANS.py：MNIST数据集转换（现在不用了）。
neuralNetwork.py：神经网络类。包括初始化，训练和查询函数。
test.py：使用神经网络，用给的测试集进行测试。
testImage.py：用自己画的数字，进行测试。
